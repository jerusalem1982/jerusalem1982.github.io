CentOS安装JDK

2019-12-18
阅读数量:次

1、卸载CentOS操作系统自带JDK环境
查找Java安装依赖库：rpm -qa | grep java
卸载Java依赖库：yum -y remove java*
2、创建存放文件的目录：mkdir /usr/lib/jvm
3、解压JDK安装包到当前目录：tar -zxvf jdk-8u231-linux-x64.tar.gz
重命名：mv jdk1.8.0_231/ jdk1.8
4、配置JDK
添加JDK全局变量
打开当前用户下的.bash_profile文件并进行编辑：vi ~/.bash_profile
添加如下内容：
export JAVA_HOME=/usr/lib/jvm/jdk1.8
export PATH=$PATH:$JAVA_HOME/bin
若要是配置的内容立即生效，则需要执行命令：source ~/.bash_profile
